# FilterJobHH — удалённые настройки

Этот репозиторий хранит файл **config.json** для расширения FilterJobHH.

## Зачем

Если hh.ru изменит вёрстку сайта — правьте `config.json` здесь. Расширение подхватит изменения в течение ~24 часов (без новой версии в Chrome Store).

## Важно

- Репозиторий должен оставаться **Public** — иначе расширение не сможет скачать файл.
- В файле нет паролей и личных данных — только технические настройки.

## Файлы

| Файл | Назначение |
|------|------------|
| `config.json` | Селекторы страницы hh.ru, пороги фильтров, списки маркеров агентств |
